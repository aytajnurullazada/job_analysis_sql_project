# Data Analyst Job Market Analysis (SQL)
This project analyzes the **data analyst job market** using SQL to identify  
**salary trends**, **in-demand skills**, and **high-value skills**.  
The analysis focuses on **remote data analyst roles** with available salary data.

---

## Business Questions
1. What are the **top-paying remote data analyst jobs**?
2. What **skills are required** for the highest-paying roles?
3. What are the **most in-demand skills** for data analysts?
4. Which skills are associated with the **highest average salaries**?
5. What are the **most optimal skills to learn** based on demand and salary?

---

## Dataset
The project uses a real-world job postings dataset containing:
- Job titles and roles  
- Salaries  
- Company information  
- Required skills  
- Remote work status  

The dataset is publicly available on **Hugging Face**:  
https://huggingface.co/datasets/lukebarousse/data_jobs  

---

## Analysis Summary

### Top-Paying Data Analyst Jobs
- Identified the **top 10 highest-paying remote data analyst positions**
- Filtered roles with valid salary data

### Skills for Top-Paying Jobs
- Analyzed which **skills are required** by the highest-paying roles

### Most In-Demand Skills
- Ranked skills based on how frequently they appear in job postings

### Top Skills Based on Salary
- Calculated the **average salary per skill**
- Identified skills linked to higher-paying roles

### Optimal Skills to Learn
- Combined **skill demand** and **average salary**
- Focused on **remote data analyst positions**
- Highlighted skills with both high demand and strong pay
